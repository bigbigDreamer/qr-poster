# 二维码海报组件

基于 `React` 的，具备 `Headless Component` 特性的海报二维码组件。  

## 背景

在常见的活动推广中，经常需要做宣传海报分享或者前台立牌做销售，我们希望可以进一步提升业务开发的效率；

传统的 `UI` 组件难以满足灵活多变且布局繁杂的二维码海报，需要一种解法“以不变应万变”；

## 目标

- 封装核心的海报生成能力、保存能力；
- 兼容性优良，满足内核低端设备；
- `Headless` 特性，不封装任何 `CSS`;
